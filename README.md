# Course 5: C++ Fundamentals & Projects

This repository contains my collection of C++ programs developed as part of my coursework. The projects are organized into two primary categories: foundational **Problems** and advanced **Projects**.

## Project Structure

The project is structured to separate individual exercises from final projects:

* `/Problems`: Contains individual C++ source files for fundamental algorithms, and logic exercises.
* `/Projects`: Contains more complex, multi-functional applications.

## Technologies Used

* **Language**: C++20
* **Build System**: CMake (Minimum version: 4.1)
* **Environment**: CLion

## Contents

### Problems

This folder includes various exercises covering:

* **Mathematical Algorithms**:
* `MultiplicationTable`: Generates multiplication tables.
* `printPrimeFrom1ToN`: Prints all prime numbers up to N.
* `PerfectNumber`: Identifies and prints perfect numbers.
* `perfectFrom1ToN`: Prints all perfect numbers from 1 to N.
* `Factorial_Like_Pattern`: Prints a factorial-like number pattern.
* `My_Absolute_Function`: Custom implementation of absolute value.
* `My_Round_Function`: Custom implementation of rounding.
* `My_Floor_Function`: Custom implementation of floor math.
* `My_Ceil_Function`: Custom implementation of ceiling math.
* `My_Sqrt_Function`: Custom implementation of square root.


* **Digit Manipulations**:
* `reverseNumber`: Reverses the digits of a given number.
* `sumOfdigits`: Calculates the sum of a number's digits.
* `returnReversed`: Returns the reversed integer.
* `digitFrequency`: Counts the frequency of a specific digit.
* `allDigitFrequencies`: Calculates frequency for all digits in a number.
* `printDigitsInOrder`: Prints digits of a number in order.
* `random_digits`: Handles operations on random digits.


* **Array Manipulations**:
* `Array`: Basic array structure.
* `Fill_Array_With_Randoms`: Populates arrays with random values.
* `Minimum_Number_In_Random_Array`: Finds the minimum value in an array.
* `Sum_Of_Randoms_In_Array`: Sums array elements.
* `Average_Of_Random_Array`: Calculates the average of array elements.
* `Copy_Array`: Duplicates an array.
* `Copy_Primes_From_Array`: Filters and copies prime numbers.
* `Sum__OfArrays`: Sums two arrays.
* `Shuffle_Array`: Randomizes array order.
* `Copy_Array_In_Reverse`: Copies an array in reverse order.
* `Fill_Array_With_Keys`: Fills arrays with generated keys.
* `Search_Array_Of_Randoms`: Search functionality for random arrays.
* `Check_For_Number_In_Array`: Boolean check for value existence.
* `Simi_Dynamic_Array`: Simulates dynamic array behavior.
* `Copy_Array_Using_AddElement`: Copying via add-element logic.
* `Copy_Odds_Using_AddElement`: Filters and copies odd numbers.
* `Copy_Primes_Using_AddElement`: Filters and copies primes using add-element logic.
* `Copy_Distinct_Numbers_From_Array`: Removes duplicates during copy.
* `Pilindrome_Array_Check`: Verifies if an array is a palindrome.
* `Count_Odds_In_Array`: Counts odd elements.
* `Count_Evens_In_Array`: Counts even elements.
* `Count_Positives_In_Array`: Counts positive numbers.


* **Other**:
* `Lucky`: Handles "Lucky" number logic.
* `PalindromeNumber`: Checks if a number is a palindrome.
* `Problem14`: Custom logic exercise.
* `latterPattern`: Prints specific letter patterns.
* `printAAAToZZZ`: Pattern printing range.
* `passwordGuess`: Password simulation/guessing logic.
* `Encryption_Decryption`: Basic string encryption logic.
* `print_3_Randoms_from_1_to_10`: Randomized output exercise.
* `Generate_Keys`: Utility to generate random keys.
